# 项目简介

定时按什么值得买的好价排行榜,收集优惠信息并推送至邮箱,基于java实现,欢迎各位值友来捉虫 
* 邮件的正文内容是html,优惠信息将以表格的形式显示在正文中
* 已推送的优惠信息会被记录在项目根目录的`pushed.txt`文件中,每次运行项目都会自动提交代码,更新这个文件
* 有其他改进的建议可以提issue给我
* 麻烦给个star支持一下:heart:

# 自定义过滤逻辑

### 标题关键词过滤

`black_words.txt`文件中可以自定义关键词,**多个关键词换行**

在优惠信息的标题中包含任意一个关键词时,就不会被推送
